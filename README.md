# Mounika Nimmanagonti Portfolio

This is a responsive tech-role portfolio built with plain HTML, CSS, and JavaScript.

## How to open in Visual Studio Code

1. Open Visual Studio Code.
2. Choose **File > Open Folder**.
3. Select `/Users/nimmanagonti05/Desktop/MYPORTFOLIO`.
4. Open `index.html`.

You can preview it by opening `index.html` in a browser. If you use the VS Code Live Server extension, right-click `index.html` and choose **Open with Live Server**.

## Recommended VS Code setup

When VS Code opens this folder, it should suggest installing these extensions:

- Live Server
- Prettier
- Auto Rename Tag
- HTML CSS Support

Install the recommendations when VS Code asks.

## Run options

### Best option

Right-click `index.html` and choose **Open with Live Server**.

### Terminal option

Open the VS Code terminal and run:

```bash
python3 -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

### VS Code task option

Press `Cmd + Shift + P`, search **Tasks: Run Task**, then choose:

```text
Open Portfolio with Python Server
```
